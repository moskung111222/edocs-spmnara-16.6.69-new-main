# Project Overview: NWT Document Submission System

NWT Document Submission System (ระบบยื่นคำขอเอกสารออนไลน์ สพม.นราธิวาส) เป็นเว็บแอปพลิเคชันสำหรับอำนวยความสะดวกในการยื่นคำขอรับเอกสารทางวิชาการและการจัดการศึกษาในสังกัดสำนักงานเขตพื้นที่การศึกษามัธยมศึกษานราธิวาส (สพม.นราธิวาส) แก่ประชาชนผ่านระบบออนไลน์แบบครบวงจร

## วัตถุประสงค์
1. **บริการประชาชน**: เพื่อให้ผู้ขอรับบริการ เช่น ประชาชน ศิษย์เก่า และนักเรียน สามารถส่งคำขอเกี่ยวกับการศึกษา (เช่น การขอใบแทนใบสุทธิ/ประกาศนียบัตร, ใบแสดงผลการเรียน, หนังสือรับรองวุฒิการศึกษา) ได้ทางออนไลน์ ตลอด 24 ชั่วโมง
2. **ปรับปรุงกระบวนการทำงาน**: เปลี่ยนแปลงระบบการส่งเอกสารแบบกระดาษมาสู่ระบบงานสารบรรณอิเล็กทรอนิกส์ เพื่อการตรวจสอบและจัดเก็บเอกสารอย่างเป็นระบบ
3. **ความปลอดภัย**: เพื่อปกป้องข้อมูลส่วนบุคคลของประชาชนด้วยระบบตรวจสอบรหัสผ่านแบบใช้ครั้งเดียว (OTP Verification) ผ่านอีเมล และแยกการจัดเก็บเอกสารนอกโฟลเดอร์ Web Root

## ผู้ใช้งาน (Target Users)
1. **ประชาชน / นักเรียน / ศิษย์เก่า (Citizen / Applicant)**: ยื่นแบบฟอร์ม แนบไฟล์หลักฐาน ติดตามสถานะ ดำเนินการแก้ไขเอกสารตามคำสั่ง และแชทติดต่อสื่อสารกับเจ้าหน้าที่
2. **เจ้าหน้าที่สารบรรณ / ทะเบียน (Staff)**: ตรวจรับคำขอ ตรวจเช็กความถูกต้องของเอกสารหลักฐาน และจดบันทึกภายในหรือพิมพ์ข้อความตอบโต้กับประชาชน
3. **ผู้อำนวยการกลุ่ม / หัวหน้าหน่วยงาน (Head)**: ตรวจทานคำขอ มอบหมายเจ้าหน้าที่เจ้าของสำนวน และอนุมัติ (Approve) หรือปฏิเสธ (Reject) ใบคำขอ
4. **ผู้ดูแลระบบ (Admin)**: สิทธิ์สูงสุดในการเข้าถึงระบบ จัดการเจ้าหน้าที่ ดูรายงานสถิติ และการตั้งค่าของเซิร์ฟเวอร์

## เทคโนโลยีที่ใช้ (Technology Stack)
- **Core backend**: PHP 7.4 (หรือสูงกว่า)
- **Database**: MySQL 5.7+ / MariaDB 10.3+ (เชื่อมต่อผ่าน mysqli)
- **Frontend structure**: HTML5, Vanilla JavaScript (ES6), CSS3, Bootstrap 5.3.0
- **Realtime capabilities**: Pusher WebSockets (ส่งข้อมูลและแจ้งเตือน Realtime)
- **UI Libraries & Plugins**: SweetAlert2 (หน้าจอเตือนและกล่องยืนยัน), FontAwesome 6 (ไอคอน), Chart.js (แสดงผลกราฟสถิติหลังบ้าน), DataTables (จัดการรายการคิวหลังบ้าน)
- **Mail Exporter**: PHPMailer (สำหรับการจัดส่งเมล OTP และความเคลื่อนไหวสถานะ)
